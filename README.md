# De Pátria para Pátria

## Funcionalidades

- Exibição de logotipo e cabeçalho com imagem e título.
- Sessão de introdução com citação e texto explicativo
- Estrutura responsiva com suporte a diferentes tamanhos de tela.
- Footer com informações de copyright.

---

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do site.
- **CSS3**: Estilização e layout
- Design responsivo (com suporte a media queries para diferentes tamanhos de tela).
